CSS è una lista di descrizioni

Le descrizioni possono essere il colore del testo, il font o la grandezza dei caratteri.
Conviene raccogliere tutte le informazioni nel selctor "div".
Il colore è una proprietà, mentre il colore assegnato si chiama valore.

Per identificare gli elementi si possonousare gli ID o "class".
Il primo è unico per ogni elemento, mentre class no e può essere usato su più elementi.
Per usare ID si usa #id, per class si usa . class (con lo spazio eh)
